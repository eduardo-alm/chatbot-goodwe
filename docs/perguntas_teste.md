# Modelo de Teste — Chatbot GoodWe

## Contexto: ChargeGrid Intelligence e EV ChargeOps

---

### Pergunta 1
**Pergunta:** O que é o ChargeGrid Intelligence?

**Resposta esperada:** O ChargeGrid Intelligence é um sistema da GoodWe para eletropostos comerciais que orquestra potência, registra ciclos de carga, realiza faturamento automático e integra comunicação entre os equipamentos.

---

### Pergunta 2
**Pergunta:** Como funciona o faturamento no ChargeGrid Intelligence?

**Resposta esperada:** O sistema registra cada ciclo de carga realizado e gera automaticamente o faturamento para o operador comercial, com base no consumo de energia de cada sessão.

---

### Pergunta 3
**Pergunta:** O que é o EV ChargeOps?

**Resposta esperada:** O EV ChargeOps é um sistema da GoodWe voltado para condomínios, que gerencia o uso compartilhado de carregadores entre moradores, com controle de acesso e faturamento individual por unidade.

---

### Pergunta 4
**Pergunta:** Como o síndico pode controlar o uso dos carregadores no condomínio?

**Resposta esperada:** Pelo EV ChargeOps, o síndico pode monitorar quais moradores estão utilizando os carregadores, definir regras de acesso e acompanhar o consumo individual de cada unidade para fins de faturamento.

---

### Pergunta 5
**Pergunta:** O que fazer quando um eletroposto não responde?

**Resposta esperada:** O técnico deve verificar a conectividade do equipamento com o sistema ChargeGrid Intelligence, checar os logs de comunicação e, se necessário, reiniciar o módulo de comunicação do eletroposto.
